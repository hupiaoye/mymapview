## 广勘智图市场定位

### 目标用户
1. **测绘从业者** - 日常看图、坐标转换
2. **规划设计师** - 数据查看、标注管理
3. **户外工作者** - 轨迹记录、位置标注
4. **数据处理人员** - 格式转换、批量处理
5. **不愿付费用户** - 替代奥维VIP功能

### 核心优势
1. ✅ **完全免费** - 无VIP、企业限制
2. ✅ **简单易用** - 现代界面，学习成本低
3. ✅ **专业坐标系** - 自定义四/七参数
4. ✅ **轻量快速** - 启动快，占用资源少
5. ✅ **专注看图** - 不做全能GIS，专注核心功能

### 差异化竞争
| 竞争对手 | 弱点 | 广勘智图优势 |
|---------|------|-------------|
| 奥维地图 | VIP收费、企业功能臃肿 | 完全免费、轻量专注 |
| LSV | 侧重三维、二维功能弱 | 二维看图更专业 |
| Bigemap | 价格高、学习成本高 | 免费易用、快速上手 |
| 91卫图 | 功能单一 | 功能综合、格式全面 |
